# 7. 页面的基础布局

根据UI设计，我们将HTML结构分为三个部分：
  1. 标题：显示当前页面的标题和日期；
  2. 菜单：可收缩的菜单，通过菜单切换路由；
  3. 内容：负责内容展示；

## 7.1 布局结构
layout.vue是整个项目的布局组件，在layout组件中唯一变化的就是`router-view`组件，是[vue-router](https://router.vuejs.org/zh-cn/essentials/getting-started.html)内置的组件，将会根据路由匹配到的组件来替换`router-view`。
>注意：在menu-bar组件中我们注册了`menuSelect事件(自定义事件)`，当用户选择菜单时，会由`menu-bar实例`触发该事件，进行收缩菜单。
这里有个父子组件的概念，父组件指的是`layout.vue`，子组件指的是`menu-bar.vue`。

`page/layout/layout.vue`
```html

<template>
```

```html
      width: 100%;
      height: 100%;
      opacity: 1;
      background: rgba(172,185,201,.4);
      transition: opacity .3s ease;
    }
  }
</style>

<template>
  <div class="layout-menu-bar">
    <el-menu
      @select="handleMenuSelect"
      class="layout-menu-bar_wrap_menu"
      mode="vertical"
      :default-active="activeMenu"
      background-color="#5b7492"
      text-color="#fff"
      active-text-color="#ffd04b"
    >
      <router-link
        :key="item.path"
        :to="item.path"
        v-for="item in menuData"
      >
        <el-menu-item :index="item.path" class="layout-menu-item">
          {{ item.name }}
        </el-menu-item>
      </router-link>
    </el-menu>
    <div class="layout-cover"></div>
  </div>
</template>

<script>
import { Row, Col, Menu, MenuItem } from 'element-ui';
import enums from '../../../common/js/enums';

export default {
  name: 'MenuBar',
  components: {
    [Menu.name]: Menu,
    [MenuItem.name]: MenuItem,
    [Row.name]: Row,
    [Col.name]: Col,
  },
  data() {
    return {
      menuData: enums.menu,
      activeMenu: '/',
    };
  },
  methods: {
    handleMenuSelect() {
      this.$emit('menuSelect');
    }
  }
};
</script>
```
